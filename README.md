Eat the frog

from user side:
A simple to-do list application which can rearrange tasks entered to fit a desired working method. The name is derived to the expression which means to do the thing you dread first to avoid prolonged procrastination. User can select only one task they deem to be their "frog". Since different methods work for differ people, the app offers other strategies to tackle tasks.
